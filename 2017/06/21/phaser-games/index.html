Drake Themed 2D-Pong/2D-Asteroids

21 Jun 2017

I wanted to dabble into game development however didn’t want to start with a heavy game engine like Unity or Unreal engine. I decided to look into web games and came across a javascript library called phaser.js.

It’s a pretty cool open source framework to make canvas and WebGL powered browser games. I decided to dabble into some sprite making and make some Drake themed classics.
